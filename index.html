Anayurt Teknoloji

Signal Processing Engineer

Developed and iteratively improved radio signal processing software for defence communications products, shipped as a cross-platform C++ library.

HF and VHF radio signals are distorted by atmospheric propagation, multipath fading, and interference. Each transmission mode requires a different demodulation and error-correction chain. The decoder library handles carrier synchronisation, symbol timing recovery, demodulation, and forward error correction across modes of widely varying complexity — from simple narrowband waveforms to multi-carrier schemes with interleaving and convolutional coding. Decoders were continuously refined using field feedback data from real-world signal captures.

Supporting tools included a Qt-based validation environment for batch test execution with bit-error-rate regression tracking across Windows and Linux builds, and an Android software-defined-radio application in Kotlin with a C++ hardware interface layer over NDK/JNI.

Key Highlight

Iterative decoder improvement driven by field feedback data across HF and VHF transmission modes of varying complexity

MATLAB C++ Qt Kotlin Android NDK JNI
